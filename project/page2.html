How It Works

  1. 1) Contact

    Send a request using the contact form. You will receive a reply with the next available times.

  2. 2) First Session

    We clarify your goals, main concerns, and context. Together we define priorities and a clear direction.

  3. 3) Plan & Progress

    Sessions focus on practical tools and measurable progress. You get exercises to apply between sessions.
