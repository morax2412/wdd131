W05 Website Planning Document

Topic: Psychologist Professional Website

This document is the site plan (blueprint) for the final project website. It defines the name, purpose, target scenarios, design system (colors + typography), and wireframes for mobile and desktop.

1. Site Name

Proposed Name

MindCare Psychology (or MindCare Psicología)

I chose this name because it communicates emotional support, professionalism, and a calm, safe space for clients.

Optional Domain Ideas

  • mindcarepsychology.com
  • mindcarepsicologia.com
  • mindcareclinic.com

Domain purchase is not required for this course, but these options match the brand.

2. Site Purpose

The purpose of this website is to present a professional profile for a licensed psychologist, explain services, build trust with visitors, and make it easy to contact or request an appointment. The site will include service descriptions, pricing guidance (if appropriate), a short FAQ, and a contact form.

Primary Goals

  • Explain who the psychologist is and what specialties they offer.
  • Help visitors decide if therapy is right for them.
  • Provide a simple way to contact and request an appointment.
  • Improve credibility with clear information, tone, and design.

3. Scenarios

Typical questions a visitor (target audience) may ask:

  • “Do you help with anxiety, stress, or relationship issues? What type of therapy do you provide?”
  • “How do I book a session and what are your available hours (online / in-person)?”
  • “How much does a session cost and how many sessions do people usually need?”
  • “Where is your office located and how can I contact you quickly?”

4. Color Scheme

The color palette is designed to feel calm, trustworthy, and warm (healthcare + mental wellness aesthetic). Primary is used in the header and headings, and Accent is used for badges and highlights in this document.

Primary: Deep Teal #2E6F6B

Use: header, headings, buttons, links, key highlights.

Accent: Warm Gold #F4B860

Use: call-to-action accents, badges, small highlights, icons.

Support Colors

  • Background: #F6F8FA
  • Body text: #1F2937
  • Borders: #E5E7EB

5. Typography

The site will use a clean, modern font pairing for readability and a professional feel. This planning document already uses these fonts to demonstrate the typography choices.

  • Headings: Poppins (600–700) — used for h1, h2, h3.
  • Body text: Inter (400–600) — used for paragraphs and lists.

6. Wireframes

Wireframes are simple layout diagrams showing content blocks for the Home page in mobile and desktop views.

Mobile View (Home)

~375px
Header: Logo + Menu (hamburger)
Hero: Short headline + 1–2 lines + CTA “Book / Contact”
Services: 3 stacked cards (Anxiety, Stress, Couples, etc.)
About: Photo + short bio + credentials
Testimonials: 2–3 short quotes (optional)
FAQ: Accordion style list (2–4 questions)
Contact: Simple form + WhatsApp/Email + Location
Footer: Nav links + “References” link

Desktop View (Home)

~1024px+
Header: Logo left + Nav right (Home, Services, About, Contact)
Hero (2 columns): Text + CTA on left, image on right
Services (3 columns): 3 service cards in a row
About (2 columns): Photo left, bio + credentials right
Social Proof: Testimonials row or stats
FAQ + Contact (2 columns): FAQ left, form right
Footer: Multiple columns + “References” link

File Naming & Structure

This plan follows the course naming conventions and folder structure for the final project.

  • project/siteplan.html
  • project/index.html
  • project/styles/site.css
  • project/scripts/main.js
  • project/references.html (linked in the footer)
  • project/images/

Notes for Development

  • All pages will be responsive using CSS media queries (mobile-first).
  • Accessibility: semantic HTML, proper headings, sufficient contrast, labels for form inputs.
  • Performance: optimized images, minimal JS, and validated HTML/CSS.
